A curated collection of high-quality prompts for AI tools — tested and refined across real use cases.

## How to use

Each `.md` file contains:
- The prompt (copy-paste ready)
- What tool it works best with
- Why it works (the technique behind it)
- Variations you can try

## Tips

- **Be specific about output format** — telling the model "respond as a markdown table" or "use bullet points" dramatically improves structure
- **Role assignment works** — "You are a senior security analyst" primes better responses than just asking the question
- **Chain prompts** — break complex tasks into 2-3 sequential prompts instead of one massive one
- **Include constraints** — "in under 200 words" or "focus only on X" prevents rambling
